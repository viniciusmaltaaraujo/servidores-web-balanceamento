📚 Sobre este Servidor Web

Detalhes Técnicos da Implementação

🐳 Tecnologias Utilizadas

  • Apache HTTP Server 2.4: Servidor web robusto e amplamente utilizado
  • Docker: Containerização para ambiente isolado e reproduzível
  • Docker Compose: Orquestração simples de containers
  • Alpine Linux: Imagem base leve e segura

⚙️ Configurações Aplicadas

  • Porta: 8080 (host) → 80 (container)
  • Volume: ./html mapeado para /usr/local/apache2/htdocs/
  • Restart Policy: unless-stopped
